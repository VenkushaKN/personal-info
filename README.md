# Personal Information Manager

## Description
This project stores and displays personal information using Python.

## Features
- Uses variables
- Takes user input
- Validates input
- Calculates age in months
- Displays formatted output

## How to Run

Step 1: Install Python

Step 2: Open terminal

Step 3: Navigate to folder

Step 4: Run command:

python personal_info.py
